## üì¶ Installation & Setup

To run this project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/QasimTareen/portfolio.git
   cd portfolio
   ```

2. **No Build Step Required**:
   This project is designed to run directly in the browser using ESM modules. You simply need a local development server.

3. **Run with VS Code Live Server**:
   - Install the "Live Server" extension.
   - Right-click `index.html` and select **"Open with Live Server"**.

   *Alternatively, using Node.js:*
   ```bash
   npx serve .
   ```
